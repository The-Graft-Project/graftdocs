## Quick Installation

Get Graft up and running in seconds.

<Tabs>
  <TabItem label="Linux & macOS">
    ```bash
    curl -sSL https://raw.githubusercontent.com/skssmd/Graft/main/bin/install.sh | sh
    ```
  </TabItem>
  <TabItem label="Windows">
    ```powershell
    winget install graft
    ```
    *Or use our [PowerShell script](/install#windows-quick-install)*
  </TabItem>
</Tabs>

## Why Graft?

<div class="features-grid">
	<Card title="Zero DSL" icon="pencil">
		The `graft-compose.yml` is a pure Docker Compose file that provides context to Graft.
	</Card>
	<Card title="Auto Infra" icon="add-document">
		Automatic setup for Traefik, SSL, shared Postgres, and Redis on any fresh server.
	</Card>
	<Card title="Agentless SSH" icon="rocket">
		Instantly start sessions or run commands on your project host without hunting for IPs.
	</Card>
	<Card title="CI/CD Built-in" icon="setting">
		Graft reads context and generates production-ready GitHub Actions workflows automatically.
	</Card>
	<Card title="Instant Rollbacks" icon="random">
		Every deployment is versioned. Revert projects or services to stable states in seconds.
	</Card>
	<Card title="DNS & Migration" icon="external">
		Automated Cloudflare DNS mapping and zero-downtime migration between servers.
	</Card>
	<Card title="Project Context" icon="list-format">
		Graft manages project-to-server connections and knows where each project lives.
	</Card>
	<Card title="Docker Passthrough" icon="seti:docker">
		Run any Docker Compose command remotely through Graft. Perfect for live debugging.
	</Card>
</div>
